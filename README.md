# 🏢 CoreApp - El Mirador

**CoreApp** es una solución integral de software diseñada para la administración eficiente y transparente del complejo residencial "El Mirador". Este sistema centraliza la gestión operativa, financiera y de convivencia, conectando a la administración con los residentes (dueños y arrendatarios) en una plataforma única, moderna y altamente escalable.

---

## 🚀 Características Principales

El sistema está dividido en dos flujos principales de usuario, garantizando una experiencia centrada y contextual:

### 🛡️ Módulo de Administración
* **Dashboard Financiero:** Visualización analítica de ingresos mensuales, tasas de morosidad y deuda total en tiempo real.
* **Directorio de Unidades:** Gestión de perfiles activos (Dueños, Residentes, Corredores) por departamento, con filtros estructurales avanzados (Torre, Piso, Estado de Ocupación).
* **Gestión de Solicitudes:** Trazabilidad completa de los requerimientos de mantenimiento y plomería de los residentes, con estados de avance y prioridades.
* **Bitácora de Auditoría:** Registro de seguridad inmutable que captura eventos críticos del sistema (Inicios de sesión, ediciones, eliminaciones) para un monitoreo transparente de las acciones administrativas.

### 🏠 Módulo de Residentes
* **Panel de Pagos:** Consulta de gastos comunes históricos y próximos vencimientos con pasarela de pago integrada.
* **Reporte de Incidencias:** Creación y seguimiento de solicitudes de soporte técnico y mantenimiento para los departamentos o áreas comunes.
* **Avisos del Conjunto:** Tablón de anuncios en tiempo real para notificaciones críticas de la comunidad.

---

## 🛠️ Stack Tecnológico

El proyecto está desarrollado bajo altos estándares de ingeniería y diseño de interfaces (UI/UX), priorizando el rendimiento y la consistencia visual:

* **Frontend:** React (Vite)
* **Lenguaje:** TypeScript (Tipado estricto para modelos de datos y estados)
* **Estilos:** Tailwind CSS (Sistema de diseño basado en utilidad y fuente Inter)
* **Gráficos:** Recharts (Visualización de datos financieros)
* **Iconografía:** Lucide React

---

## 📐 Metodología y Arquitectura

El desarrollo de *CoreApp - El Mirador* fue gestionado utilizando marcos de trabajo ágiles (SCRUM) y documentado exhaustivamente mediante un Documento de Arquitectura de Software (DAS). La interfaz gráfica fue iterada tras rigurosas pruebas de Evaluación Heurística (Nielsen) para garantizar una usabilidad de nivel profesional y la prevención de errores en flujos críticos.
